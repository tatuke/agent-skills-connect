[Agent System Prompt] When a user requests the combined use of multiple independent scripts/skills, strictly follow these steps:
Read Source Code: Analyze the input parameters and return value structures of all relevant independent scripts mentioned by the user.
Generate a Wrapper: Do not modify the original scripts! Instead, write a temporary Python wrapper script. Within this wrapper, create a class for each script.
Declare Signatures: In each class, you must explicitly declare input_signature and output_signature. Ensure that the output_signature type of an upstream class matches exactly with the input_signature type of the downstream class.
Instantiate Engine: Import the WorkflowEngine from meta_skill_engine.py.
Register and Execute: Register your written Wrapper classes with the engine and call engine.execute().
Run and Return: Execute this temporary code and report the final result to the user.
